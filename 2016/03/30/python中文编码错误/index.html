Python中文编码错误(UnicodeEncodeError)

  • Mac
Debug 
UnicodeEncodeError: ‘ascii’ codec can’t encode

解决方案1：

import sys
reload(sys)
sys.setdefaultencoding('utf-8')

解决方案2:

不使用open打开文件，而使用codecs.open(), 在输出中文文本时，采用UTF-8编码：

f = codecs.open(‘XX.txt’, ‘a’, ‘utf-8’)
f.write(txts)
f.close()
